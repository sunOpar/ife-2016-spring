# ife-2016-spring

## ife介绍

ife全名为：百度前端技术学院
是一个面向大学生的前端技术学习平台，当然即使是非在校生也能参加。
主要是由百度创办的免费的前端技术学习实践、交流、分享平台。

ife精心设计了许多编码任务，让学生可以循序渐进、由浅入深地在实践中掌握技术；并且还致力于帮助大家建设一个更好的技术交流与分享氛围。
